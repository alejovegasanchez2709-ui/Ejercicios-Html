Diccionario de Desarrollo Web

HTML: Es el lenguaje de marcado estándar para la creación de páginas web. Permite definir la estructura y el contenido del documento de forma lógica.

CSS: Es un lenguaje de diseño gráfico que se utiliza para definir la presentación de un documento. Con CSS, puedes cambiar el color, la fuente y el diseño de tu sitio.

JavaScript: Es un lenguaje de programación que permite crear contenido dinámico e interactivo en las páginas web, como mapas, animaciones o formularios.
